## 📌 Overview

Heart murmurs are abnormal heart sounds that may indicate cardiovascular diseases.
This project presents a Deep Learning based solution using Convolutional Neural Networks (CNN) for automatic heart murmur detection from Phonocardiogram (PCG) signals.

The complete pipeline includes:

- Signal preprocessing
- Noise removal
- Feature extraction
- CNN model training
- Performance evaluation

## ⚙ Installation

Clone the repository

```bash
git clone git@github.com:ronakmahlawat/Heart-Murmur-Detection-CNN.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```

Open

```
CNN.ipynb
```
